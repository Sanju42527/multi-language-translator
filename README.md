# Multi Language Translator

[![GitHub Pages](https://img.shields.io/badge/Deployed%20on-GitHub%20Pages-blue)](https://sanju42527.github.io/multi-language-translator/)

## Overview

Multi Language Translator is a web-based application that enables seamless translation of text across multiple languages. Built with modern web technologies, it provides an intuitive interface for quick and efficient language translation.

## Features

- **Multi-Language Support**: Translate text across 22+ Indian languages
- **Auto-Detection**: Automatically detect the source language
- **User-Friendly Interface**: Clean and intuitive design for easy navigation
- **API Integration**: Powered by Sarvam AI translation API
- **Fast Translation**: Quick and reliable translation results
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Advanced Options**: Additional customization features for advanced users

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Sarvam AI API Key (Get one at [https://sarvam.ai/](https://sarvam.ai/))

### Usage

1. Visit the application at [https://sanju42527.github.io/multi-language-translator/](https://sanju42527.github.io/multi-language-translator/)
2. Enter your Sarvam AI API Key in the input field
3. Enter the text you want to translate
4. Select the source language (Auto Detect by default)
5. Select the target language
6. Click the "Translate" button
7. View the translation result below

## API Configuration

To use this application, you need:

1. A Sarvam AI account
2. A valid API Key from Sarvam AI
3. Enter the API Key in the application interface

Get started with Sarvam AI: [https://sarvam.ai/](https://sarvam.ai/)

## Deployment

This project is deployed on GitHub Pages and automatically builds from the main branch.

### Deploy to Your Own GitHub Pages

1. Fork this repository
2. Go to Settings ‚Üí Pages
3. Select "Deploy from a branch"
4. Choose the main branch and root folder
5. Click Save
6. Your site will be available at `https://yourusername.github.io/multi-language-translator/`

## Troubleshooting

### API Key Issues
- Ensure your API key is valid and active
- Check that you have sufficient API quota
- Verify your internet connection

### Translation Not Working
- Verify the API key is correctly entered
- Check the browser console for error messages
- Ensure both source and target languages are selected
- Try a simpler text input
